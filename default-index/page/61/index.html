常用网站汇总

Posted on 2022-07-31 | In 工具

地震学常用网站

数据获取

  • IRIS ：免费的、全球性的学习性地震资料网站，SAC、Rdseed等软件均可在此网站中下载
  • PASSCAL：PASSCAL软件包网站
  • GMT ：GMT地震波绘图软件
  • 美国联邦地质调查局 ：全球地震资料。但是网站需要梯子
  • globalCMT ：具有震源机制解的地震数据
  • MDA：查询所有台网和台站信息（不推荐）
  • GMAP：在地图上设置条件显示地震台站。
  • FIND_EVENT：限制条件查找地震事件。点击下方的事件名称可以进入查找台站页面，十分推荐！！！
  • PyWEED：波形数据获取，申请简单，基于Python。
  • SeismiQuery：已失效，但是该网址提供了其他数据获取网站（包括上述介绍的一些网站）
  • 日本防灾科研——Hi-net：日本数据下载，但是需要注册。
  • 德国台站数据下载：印度、东南亚、南美等地区台站。

数据处理

  • seisan：地震学数据处理软件
  • Obspy：地震学处理Python包
  • SAC：最常用的地震学数据处理软件
  • Rdseed：数据转换软件
  • ISPAQ：现场数据质量分析
  • mseed2sac：数据格式转换软件，类似Rdseed
  • TAUP：安装教程及相关教程见地震“学”网站
  • PDCC：数据管理软件
  • PQL：数据波形信息与简单处理（适用于野外即时处理）。安装教程。

学习网站

  • 地震”学“网站 ：出色的地震学学习网站
  • Seisman ：出色的地震学相关学习博客，与地震“学”网站相同
  • SAC中文文档：学习SAC必备中文参考书
  • GMT中文社区：GMT学习官方中文社区
  • mseed2sac手册：Seisman博客中的教程，中文教程

常用软件

编程与编辑器

  • Sublime ：编辑器，但是推荐Pycharm
  • Pycharm ：Python推荐编辑器
  • Anaconda ：可批量管理Python库
  • CodeBlocks ：C语言推荐编译器
  • Python
  • Git ：Github工具
  • Matlab ：数学计算与绘图工具
  • Octave:类似与Matlab的数学计算软件
  • Octave_Online:类似与Matlab的数学计算软件

文本编辑器、论文编辑工具

  • Node.js ：博客工具
  • writage ：Word转Markdown
  • pandoc ：Word转Markdown工具
  • libreoffice ：办公套件，类似于office
  • Latex ：论文排版软件
  • LYX：类似Latex的论文排版软件

其他网站

  • hexo ：博客搭建框架
  • 菜鸟教程 ：较为齐全的编程教程网站
  • BGI ：全球重力数据网站
  • ubuntu ：Ubuntu官方网站
  • ubuntukylin ：优麒麟
  • deepin ：深度操作系统
  • Centos ：Centos
  • Markdown ：Markdown语法官方网站
