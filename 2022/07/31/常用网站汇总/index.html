常用网站汇总

Posted on 2022-07-31 Edited on 2022-08-03 In 工具

地震学常用网站

  • IRIS ：免费的、全球性的学习性地震资料网站，SAC、Rdseed等软件均可在此网站中下载

  • GMT ：GMT地震波绘图软件

  • 美国联邦地质调查局 ：全球地震资料。但是网站需要梯子

  • globalCMT ：具有震源机制解的地震数据

常用软件

  • Sublime ：编辑器，但是推荐Pycharm

  • Pycharm ：Python推荐编辑器

  • Anaconda ：可批量管理Python库

  • CodeBlocks ：C语言推荐编译器

  • Python

  • Git ：Github工具

  • Matlab ：数学计算与绘图工具

  • Node.js ：博客工具

  • writage ：Word转Markdown

  • pandoc ：Word转Markdown工具

  • libreoffice ：办公套件，类似于office

  • Latex ：论文排版软件

其他网站

  • hexo ：博客搭建框架
  • 菜鸟教程 ：较为齐全的编程教程网站
  • BGI ：全球重力数据网站
  • ubuntu ：Ubuntu官方网站
  • ubuntukylin ：优麒麟
  • deepin ：深度操作系统
  • Centos ：Centos
  • 地震”学“网站 ：出色的地震学学习网站
  • Seisman ：出色的地震学相关学习博客，与地震“学”网站相同
  • Markdown ：Markdown语法官方网站
