振型文件读取

振型文件指由MINEOS min_bran输出的文件，一般命名为模型名称+_振型类型，振型类型为S、T或R。

函数

put_freq_in_same：将MINEOS输出的振型文件的一个参数整合为一个列表，按照n、l排列

read_mineos_modefile：读取MINEOS输出的振型文件中的相关信息并输出

put_0_velocity：取同一个径向阶数值的相关参数。
